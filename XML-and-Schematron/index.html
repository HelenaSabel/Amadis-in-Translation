Using the Template

Editing …

  1. Find the file odd_quickstart_template.xml in the root level of your ODD_exercise/ directory.
  2. Make a copy of it (named whatever you want) in the same directory!
  3. Open the copy in oXygen.
  4. OPTIONAL: put the right metadata in the <teiHeader> (after all, your document was not written by someone named “[you!]” :-)
  5. OPTIONAL: Write (multiple chapters of) prose documentation in the <body>.
  6. Edit the fields surrounded by square brackets, or delete entire specification elements in the <schemaSpec>.

Save early and often; check validity early and often.

If you’d like validate locally without needing access to the web (or because you’ve modified the odds4odds schema by changing its ODD file or build process), change the href="http://www.wwp.northeastern.edu/outreach/seminars/_current/handouts/ODD_exercise/schemas/odd4odds part of each of the two xml-model processing instructions (on lines 2 & 3) to href="schemas/odd4odds.
